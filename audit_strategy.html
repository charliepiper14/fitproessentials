Critical Monitoring Metrics

  • CPA Threshold: Alert if any campaign exceeds $15 for 3 days
  • Frequency Ceiling: Pause ad set if frequency exceeds 4.0
  • CTR Floor: Pause creative if CTR falls below 0.8%
  • Customer Mix: Target minimum 40% new customers
